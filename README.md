# 🪄 Magic Bone for Blender

![Blender Version](https://img.shields.io/badge/Blender-4.5%2B-orange?style=for-the-badge&logo=blender)
![Python](https://img.shields.io/badge/Python-3.11+-blue?style=for-the-badge&logo=python)

**Magic Bone** — это мощный аддон для ускорения процесса риггинга. Он автоматизирует создание костей на основе геометрии меша.

---

## 🔥 Основные возможности
* **Bones from Normals:** Создает кости, направленные строго по нормалям выбранных вершин.
* **Bones by Axis:** Быстрое создание костей вдоль осей X, Y или Z (положительных или отрицательных).
* **Between Vertices:** Создает кость точно между двумя выбранными вершинами.

---

## 🛠 Установка
1. Скачайте файл `magic_bone.py`.
2. В Blender перейдите в **Edit > Preferences > Add-ons**.
3. Нажмите **Install...** и выберите скачанный файл.
4. Активируйте чекбокс **Magic Bone**.
5. Панель появится в 3D Viewпорт справа (клавиша **N**) во вкладке **Magic Bone**.
